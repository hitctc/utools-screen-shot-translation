截屏翻译钉图

把截图直接翻译成一张可钉在桌面的图片。

这是一款放在 uTools 里的效率插件，适合看外文截图、教程图片、界面说明、产品文档时使用。你不用在截图、翻译、保存、找图之间来回切换，截完就能翻，翻完就能钉在屏幕上继续对照看。

核心功能：
截屏翻译钉图：截图后直接翻译并钉图
钉图记录：集中查看历史结果，支持预览、删除、重钉图
设置：统一管理翻译凭证、保存目录、主题和删除确认
支持保存翻译结果，后续可随时回看和复用
钉图支持缩放、复制图片、快捷关闭

快捷键：
Cmd/Ctrl + C：复制当前钉图
ESC：关闭当前钉图或关闭图片预览
鼠标滚轮：缩放钉图
macOS 双指缩放：自然缩放钉图

为什么值得用：
少一步步手动处理截图翻译
少切窗口，翻译结果可直接留在桌面上
历史记录集中管理，不用反复重新截图
预览、删除、重钉图都能在一个页面内完成

当前已支持：
截屏翻译钉图
历史记录管理
大图预览
重钉图
删除记录
主题切换
